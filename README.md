# prh Language Server

[Language Server](http://langserver.org/) of [prh](https://github.com/prh/prh).

## How to build

```
$ npm run vscode:update
$ npm run build
$ npm run test
```

## How to release

see https://code.visualstudio.com/docs/extensions/publish-extension

```
$ npm install -g vsce
$ vsce publish patch
```
